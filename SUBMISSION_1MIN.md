# NotesHub - 1 Minute Submission Summary

## What is NotesHub?

A community-driven web application for students to **upload, share, rate, and discover study notes**.

---

## Key Features ⭐

✅ **User Authentication** - Sign up, login, user profiles  
✅ **Note Upload** - Share study materials instantly  
✅ **5-Star Rating System** - Rate community notes  
✅ **Leaderboard** - Top contributors ranked by quality  
✅ **Save Favorites** - Bookmark important notes  
✅ **Light/Dark Theme** - User preference toggle  
✅ **Responsive Design** - Works on mobile & desktop  

---

## Tech Stack 🛠️

- **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
- **Storage:** localStorage + IndexedDB
- **Icons:** Font Awesome 6.4.0

---

## Project Structure 📁

```
8 HTML pages + JavaScript + CSS
├── index.html → Home/Browse notes
├── login.html → User authentication
├── signup.html → New user registration
├── upload.html → Share new notes
├── profile.html → User statistics
├── saved.html → Favorite notes
├── leaderboard.html → Top contributors
└── script.js + style.css → Logic & Design
```

---

## How to Use 🚀

1. **Open** `index.html` in browser
2. **Sign up** with username & password
3. **Upload** notes or **Browse** community notes
4. **Rate** notes with stars
5. **Save** favorites
6. **View** Leaderboard stats

**No installation or backend needed!**

---

## Why It Works ✨

- **Client-side only** - No server required
- **Instant feedback** - Real-time updates
- **Persistent storage** - Data saved locally
- **Offline ready** - Works without internet (after first load)
- **Lightweight** - ~65KB total size
- **Fast** - Loads in < 1 second
